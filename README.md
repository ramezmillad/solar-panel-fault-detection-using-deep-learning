solar-panel-fault-detection-using-deep-learning
Solar Panel Fault Detection (YOLOv8)  Streamlit app for classifying solar panel conditions using a YOLOv8 classification model.
Features - Single image classification - Batch analysis - Confidence visualization - Camera snapshot support 
Model YOLOv8m-cls trained on 875 solar panel images.  
Deployment Hosted on Streamlit Cloud.
